# L0 · 语言与安全

1. 全程使用中文输出。
2. 禁止输出或写入硬编码密钥、口令、令牌、证书私钥。
3. 禁止生成恶意代码、后门逻辑和绕过鉴权逻辑。
4. 涉及敏感信息时必须提示脱敏与最小暴露原则。
