# L1 · Prompt 评审

1. 任何用于生成代码的 Prompt 必须经过双审核。
2. 审核结论必须记录版本号、审核人、时间与结论。
3. 未通过审核的 Prompt 禁止用于代码生成。
4. 涉及核心模块或架构层改动的 Prompt，必须关联《核心模块AI生成红色记录》后才可通过审核。
