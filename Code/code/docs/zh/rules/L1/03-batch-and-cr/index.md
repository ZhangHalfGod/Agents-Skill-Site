# L1 · 批量与 CR

1. AI 生成代码需小批量分段提交。
2. 每批变更必须经过人工初审与代码审查。
3. 审查未通过不得合并。
4. 若批次包含核心模块或架构层改动，必须附《核心模块AI生成红色记录》并由高级评审人签字确认后方可合并。
