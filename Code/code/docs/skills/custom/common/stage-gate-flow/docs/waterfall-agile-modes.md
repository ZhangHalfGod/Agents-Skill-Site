# 瀑布 / 敏捷模式细则

> 外置长文；默认执行只需主技能页清单。

## 瀑布模式

- 严格按「需求分析 → 设计 → 生成 → 测试 → 上线」阶段执行
- 前一阶段未审核通过，禁止进入下一阶段
- 输出物必须包含：**阶段标识**、**审核状态**、**责任人**
- 涉及核心层/网关层/安全模块/架构代码的生成任务，阶段输出必须追加《核心模块AI生成红色记录》

## 敏捷模式

- 按 Sprint 周期执行
- 每日站会同步 AI 生成进度
- 迭代回顾时优化 Prompt
- 输出物必须包含：**阶段标识**、**审核状态**、**责任人**
- 涉及核心层/网关层/安全模块/架构代码的迭代项，必须在当期 Sprint 中补齐《核心模块AI生成红色记录》
